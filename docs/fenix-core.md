# Sistema Fénix — Núcleo del sistema

## ¿Qué es el Sistema Fénix?

El Sistema Fénix es un método de procesamiento estructurado de material de cátedra. No es un chatbot, no es un generador de resúmenes, no es una herramienta de IA genérica.

Es un **sistema** con un protocolo definido que transforma documentos académicos en material de estudio real.

---

## El problema que resuelve

Los estudiantes universitarios tienen:
- PDFs sueltos, mal escaneados, con información desorganizada
- Parciales que requieren comprensión real, no memorización de texto
- Tiempo limitado para estudiar todo el material

Las herramientas de IA genéricas ofrecen:
- Resúmenes de documentos sueltos (sin contexto global)
- Texto plano sin estructura de estudio
- Información sin jerarquía ni puntos críticos marcados

---

## La solución: procesamiento con coherencia global

### Protocolo de entrada

```
programa → [archivo]
unidad 1 → [documentos]
unidad 2 → [documentos]
unidad N → [documentos]
```

El programa de la cátedra es el mapa. Sin él, no hay coherencia.

### Qué produce el sistema

Para cada unidad:
- Conceptos jerarquizados (central / secundario / detalle)
- Tablas comparativas y esquemas
- Conexiones explícitas con otras unidades
- Puntos de examen frecuentes marcados
- Preguntas de autoevaluación

---

## Diferenciadores clave

| Aspecto | IA genérica | Sistema Fénix |
|---------|-------------|---------------|
| Alcance | Documento suelto | Programa completo |
| Coherencia | Ninguna | Global entre unidades |
| Estructura | Texto plano | Material de estudio |
| Personalización | Ninguna | Adaptado a la cátedra |
| Puntos críticos | No identificados | Marcados explícitamente |

---

## Servicios

1. **Procesamiento de cátedra**: Transformación del material completo
2. **Clases de IA**: Enseñanza 1 a 1 de herramientas de IA aplicadas
