# 🔥 fenix-web

Landing page del **Sistema Fénix** — procesamiento de documentos de cátedra e inteligencia artificial aplicada.

## Stack

- HTML / CSS / JS vanilla (sin frameworks)
- Sin dependencias externas
- Sin build step
- Deploy: Cloudflare Pages

## Estructura

```
/web          → Landing page (index.html + styles + scripts)
/data         → Documento procesado de ejemplo
/docs         → Documentación del sistema
/config       → Configuración de deploy
CHECKPOINT.md → Estado del proyecto
```

## Deploy en Cloudflare Pages

1. Conectar este repo en https://pages.cloudflare.com
2. Build command: (vacío)
3. Build output directory: `web`
4. Deploy
