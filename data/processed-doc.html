Unidad 1: Nomenclatura IUPAC

Concepto central: La nomenclatura IUPAC es el sistema estándar internacional para nombrar compuestos orgánicos. Dominarla es indispensable para leer e interpretar las reacciones de las unidades siguientes.

Reglas base para alcanos

La cadena principal es la más larga que contiene el mayor número de carbonos. La numeración comienza desde el extremo más cercano al primer sustituyente.

N° CarbonosPrefijoNombre baseFórmula
1Met-MetanoCH₄
2Et-EtanoC₂H₆
3Prop-PropanoC₃H₈
4But-ButanoC₄H₁₀
5Pent-PentanoC₅H₁₂
6Hex-HexanoC₆H₁₄
⚡ Punto de examen frecuente: En caso de empate al elegir cadena principal (mismo número de carbonos), se elige la que tiene más sustituyentes. Ante otro empate, se aplica orden alfabético.

Unidad 3: Reacciones Orgánicas

🔗 Conexión con U2: Cada reacción de esta unidad parte de los grupos funcionales identificados en la Unidad 2. Conocer el grupo funcional permite predecir el tipo de reacción posible.

Reacciones de oxidación-reducción

Las reacciones redox en química orgánica implican cambios en el estado de oxidación del carbono. Un carbono se oxida cuando pierde H o gana O; se reduce cuando gana H o pierde O.

ReacciónReactivoProductoAgente
Oxidación suaveAlcohol 1°AldehídoPCC / CrO₃
Oxidación fuerteAlcohol 1°Ácido carboxílicoKMnO₄
OxidaciónAlcohol 2°CetonaK₂Cr₂O₇
ReducciónAldehído/CetonaAlcoholNaBH₄ / LiAlH₄
⚡ Punto de examen frecuente: Los alcoholes terciarios NO se pueden oxidar con agentes suaves (no tienen H en el carbono con –OH). Este es un error clásico en parciales.

Unidad 4: Síntesis Orgánica

🔗 Conexión global: La síntesis integra todo lo visto: aplicás nomenclatura (U1), identificás grupos funcionales (U2) y seleccionás reacciones (U3) para diseñar una ruta sintética.

Estrategia de análisis retrosintético

Ante una molécula objetivo, trabajamos hacia atrás: ¿qué precursor + reacción genera este producto? Este proceso se repite hasta llegar a materiales de partida accesibles.

Metodología Fénix para síntesis: 1) Identificar el grupo funcional del producto. 2) Buscar en tabla de reacciones cuál lo genera. 3) Identificar el precursor. 4) Verificar condiciones de reacción. 5) Escribir la ruta completa.
